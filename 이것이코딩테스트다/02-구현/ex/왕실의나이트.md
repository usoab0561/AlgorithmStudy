# 8x8에서 갈 수 있는 좌표는? 나이트는 수평2칸+수직1칸 or 수평1칸+수직2칸만 가능.

```python
#...

row = input()
col = input()

steps = [(-2,1), (-1,-2), (1,-2), (2,-1), (2,1), (1,2), (-1,2), (1,-2)] # 이렇게 갈 수 있는 step을 지정해두면 간편

result = 0
for step in range(steps):
  next_row = row + step[0]
  next_col = col + step[1]
  
  if next_row >= 1 and next_row <=8 and next_col >= 1 and next_col <= 8:
    result +=1 
    
print(result)

```
