## 수 자료형의 연산

```python
a = 7
b = 3

# 나누기
print(a/b) # 2.3333333

# 몫
print(a//b) # 2

# 자리수 표현
print( format(a, ".2f") ) # 7.00 2자리수까지 표현


# 거듭제곱
print(a ** b) // 7 * 7 * 7

```
