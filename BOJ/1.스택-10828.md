```python
import sys
N = int(input())
stack = []

for _ in range(N):
  # command = input().split() 
  # 이렇게하면 반복문에서 시간초과 가능 
  command = sys.stdin.readline().split()
  if command[0] == "push":
    stack.append(command[1])
  elif command[0] == "pop":
    if len(stack) > 0:
      print(stack.pop())
    else:
      print(-1)
  elif command[0] == "size":
    print(len(stack))
  elif command[0] == "empty":
    if len(stack) == 0:
      print(1)
    else:
      print(0)
  elif command[0] == "top":
    if len(stack) == 0:
      print(-1)
    else:
      print(stack[-1])
 ```

input().split() 하는 것 보다 sys.stdin.readline().split()이 빠르다!
